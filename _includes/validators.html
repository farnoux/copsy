Define a validator

A validator represents a specific test that will be applied to an input element.

Use $.copsy.addValidator(name, fn) to define a new validator, where:

  • name is the name of this validator. If another validator is already defined with the same name, it will be overriden.
  • fn is a function that takes the form function (value[, element]) {}, where:
    • value is the value of the input field element to validate.
    • element is the input field element itself. This argument is optional.

{% highlight js %} $.copsy.addValidator("mandatory", function (value) { return value.length > 0; }); {% endhighlight %}

Note that Copsy already comes with the following validators: mandatory, numeric and email.

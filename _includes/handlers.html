Add a validation handler

Handlers allow you to define what's happenning when validation succeed or fail.

Use $.copsy.addHandler(fn) to define a new handler, where:

  • fn is a function that takes the form function (promise) {}, where promise is a jQuery $.Deferred's Promise object.

{% highlight js %} $.copsy.addHandler(function (promise) { promise.fail(function (element) { element.css('background-color', 'red'); }) .done(function (element) { element.css('background-color', 'white'); }); }); {% endhighlight %}

jquery.copsy.js

Copsy is an event-driven form validation plugin for jQuery. It is not bound to any specific tooltip plugin. For now tooltip display is based on Tipsy.

Download

You can download this project in either zip or tar formats, or also clone the project with Git by running: 

git clone git://github.com/farnoux/copsy

Demo




$('#form-color').copsy(function (when) {
  when.fail(function (e) {
    e.css('background-color', 'red');
  });
  when.done(function (e) {
    e.css('background-color', 'white');
  });
});
